# 🚀 AI SaaS Application (PERN Stack)

A **full-stack AI SaaS application** built using the **PERN stack (PostgreSQL, Express, React, Node.js)** with **subscription billing** and **secure authentication**.  
It integrates **Cloudinary**, **Clipdrop**, and **Google Gemini API** to provide multiple **AI-powered tools** for generating text, images, and resume analysis.  

---

## ✨ Key Features

- 🔐 **Authentication** – Sign-up, sign-in & profile management with [Clerk](https://clerk.com)  
- 💳 **Subscription Billing** – Premium plan access to AI features  
- 🗄️ **Database** – Serverless PostgreSQL powered by [Neon](https://neon.tech)  
- ☁️ **Cloudinary** – Store and serve images via cloud URLs  
- 🎨 **Clipdrop API** – Image generation, background removal & object removal  
- 🧠 **Google Gemini API** – AI-powered text generation for articles, blogs, and resume analysis  

---

## 🤖 AI Tools

- 📝 **Article Generator** → Generate full articles using **Gemini API**  
- 📰 **Blog Title Generator** → Create engaging blog titles with **Gemini API**  
- 🎨 **Image Generator** → Generate AI-based images using **Clipdrop**  
- 🪄 **Background Remover** → Remove background from images using **Clipdrop**  
- ✂️ **Object Remover** → Remove unwanted objects from images with **Clipdrop**  
- 📄 **Resume Analyzer** → Get AI-powered resume feedback using **Gemini API**  

---

## 🛠️ Tech Stack

**Frontend:** React  
**Backend:** Node.js + Express  
**Database:** PostgreSQL (Neon serverless)  
**Authentication:** Clerk  
**Payments:** Stripe (or similar)  
**Cloud Storage:** Cloudinary  
**AI Services:** Clipdrop API & Google Gemini API
